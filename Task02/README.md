# Task02 — Веб-приложение «Калькулятор»

Веб-версия игры из `Task01`, реализованная на стандартных возможностях PHP с использованием SQLite и паттерна Page Controller.

## Структура

- `Task02/public` — корень сайта (доступен из браузера)
- `Task02/db` — файл базы данных SQLite (вне public)
- `Task02/src` — логика приложения
- `Task02/views` — PHP-шаблоны (используется альтернативный синтаксис)

## Запуск

1. Перейти в каталог `Task02`.
2. Запустить встроенный веб-сервер PHP:

   ```powershell
   php -S localhost:3000 -t public
   ```

3. Открыть в браузере:
   - `http://localhost:3000/` — игра
   - `http://localhost:3000/history.php` — история партий

База данных создается автоматически при первом обращении к приложению: `Task02/db/calculator.sqlite`.
